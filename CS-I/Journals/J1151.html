2.1 Why is the first Jump instruction conditional? What determines whether the jump will be executed or not?
Because we want to be able to run the alternate path if a condition is false, otherwise we would always get the consequent outcome even when not wanted.
2.2: What is the purpose of the Jump instruction after the consequent?

2.3: Why is there no need for a corresponding Jump instruction after the alternative?
